# SmartBI Public Safety Boundary

## 可以公开

- 工具链设计。
- CLI 命令形状。
- fake config 和 fake output。
- 错误码设计。
- 离线校验思路。
- 写回前的审批和预演原则。

## 不可以公开

- SmartBI 地址、租户、报表 ID、内部字段口径和真实路由表。
- 用户名、密码、Cookie、Token、Session。
- 导出的 Excel、CSV、JSON、截图、录屏和日志。
- 成本、营收、利润、订单、客户、学生、家长、老师等真实数据。

## 推荐处理

| 场景 | 做法 |
| --- | --- |
| 需要复用脚本 | 先抽象成 fake demo，再进入公开仓库 |
| 需要展示成果 | 写脱敏摘要和方法，不放原始输出 |
| 需要协作排查 | 在私有仓库或本机目录处理 |
| 需要写回数据 | 必须有审批、预演、锁定和回滚方案 |
