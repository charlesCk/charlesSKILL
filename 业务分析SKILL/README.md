# 业务分析SKILL

该目录保存与业务分析、经营复盘、数据路由和研究方法有关的公开安全资产。

## 项目清单

| 项目 | 用途 | 入口 |
| --- | --- | --- |
| KOL Research Analysis | 将 KOL 历史材料学习方法沉淀为可复用的分析框架，不提交原始人名、报价、周会细节和内部材料。 | [`kol-research-analysis/`](./kol-research-analysis/) |
| SmartBI Weekly Reporting | 将 SmartBI 周报数据工具沉淀为可复用的导出、路由、校验和安全边界方法，不提交真实 BI 导出、内部路由表或凭据。 | [`smartbi-weekly-reporting/`](./smartbi-weekly-reporting/) |

## 安全边界

- 只保存方法、字段框架、运行边界、公开安全示例。
- 不保存真实 BI 导出、飞书原文、会议纪要、人名清单、报价、成本、营收、投放明细或截图。
- 需要复盘真实业务时，在本机私有目录或对应私有仓库运行。
