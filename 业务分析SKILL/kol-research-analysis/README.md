# KOL Research Analysis

## 项目定位

该项目来自今天整理的私有仓库 `KOL-`。在 `charlesSKILL` 中只保留公开安全的分析框架，用于说明如何把 KOL 历史材料、投放复盘、素材表现和后链路数据转成可复用的经营判断。

原始材料涉及内部飞书目录、周会内容、KOL 报价、投放成本、人员信息和业务数据，不适合放入公开仓库。

## 可复用价值

- 把分散在飞书、周会、表格、素材和历史复盘里的信息整理成统一分析链路。
- 用同一套维度评估 KOL：素材资产、授权范围、获客成本、约课率、到课率、转化率、GMV/Leads、ROI 和后端承接。
- 区分“能拿量”“能约课”“能转化”“能复投”四类价值，避免只看曝光或单条素材表现。
- 将跨区域表现拆开看，避免把港澳、台湾、新加坡等市场混在一个结论里。

## 使用方式

1. 在私有目录中读取真实飞书、表格、视频素材和 BI 数据。
2. 按 [`references/analysis-framework.md`](./references/analysis-framework.md) 做结构化判断。
3. 对外只输出脱敏摘要、项目阶段结论和下一步动作。
4. 任何真实人名、报价、成本、投放数据、会议原文都留在私有空间。

## 不提交内容

- 飞书目录链接、文档正文、表格导出和截图。
- KOL 真实报价、合同条件、授权细节和商务谈判记录。
- 内部人员姓名、老师名单、客户/学生/家长信息。
- 真实投放成本、营收、利润、GMV、订单或转化明细。
