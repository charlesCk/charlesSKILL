# 舆情监控SKILL

该目录存放公开平台舆情、评论、用户反馈采集相关的 Codex Skills。

| Skill | 适用平台 | 入口 |
| --- | --- | --- |
| Facebook Opinion Crawler | Facebook 帖子、视频、Reels、主页、小组讨论 | [`facebook/SKILL.md`](./facebook/SKILL.md) |
| Xiaohongshu Opinion Crawler | 小红书 / Rednote 笔记和评论 | [`小红书/SKILL.md`](./小红书/SKILL.md) |
| App Store Opinion Crawler | Apple App Store / iTunes RSS | [`appstore/SKILL.md`](./appstore/SKILL.md) |

## 目录边界

这里只保存可复用 skill、脚本、参考规则和示例配置。真实评论导出、截图、浏览器缓存、平台账号登录态和未脱敏业务数据都不能提交。
